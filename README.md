> A simulation framework for **Federated Learning across satellite networks**.

## 📌 Overview

FedSat AI is a simulation framework designed to implement **Federated Learning** in satellite edge networks. The project focuses on decentralized learning across satellite nodes, ensuring data privacy and enhancing performance through a federated approach. Utilizing **PyTorch** for training and testing models, the framework facilitates communication and data sharing while maintaining security and efficiency across satellite networks.

---

# ✨ Features

- **Federated Learning Simulation**: Distributed model training across satellite nodes.
- **PyTorch Integration**: Model creation, training, and evaluation with PyTorch.
- **Decentralized Data Processing**: Enables privacy-preserving learning without sharing raw data.
- **Satellite Network Modeling**: Simulates satellite communications for federated learning.
